# VS2017实现Udp socket多线程通信（C++）

## 资源介绍

本仓库提供了一个基于Visual Studio 2017实现的UDP服务器与客户端多线程通信程序。该程序使用C++编写，并通过`select`机制实现了多线程通信。程序已经过编译和测试，确保其稳定性和可靠性。

## 功能特点

- **UDP通信**：实现了基于UDP协议的服务器与客户端之间的通信。
- **多线程处理**：通过`select`机制实现了多线程处理，提高了通信效率。
- **Visual Studio 2017**：程序基于Visual Studio 2017开发，确保与该开发环境的兼容性。
- **已编译测试**：程序已经过编译和测试，可以直接使用。

## 使用说明

1. **下载资源**：从本仓库下载资源文件。
2. **导入项目**：将下载的文件导入到Visual Studio 2017中。
3. **关闭SDL检查**：在项目属性中关闭SDL检查，以确保程序能够正常编译和运行。
4. **编译运行**：编译并运行程序，启动服务器和客户端进行通信测试。

## 注意事项

- 请确保在Visual Studio 2017中关闭SDL检查，否则可能会导致编译错误。
- 程序已经过测试，但在不同环境下可能会有不同的表现，建议在使用前进行适当的测试。
